# Phishing Alert Analysis – SOC Level 1

## Lab Type
Blue Team / SOC Analyst (Beginner – Practical)

## Platform
TryHackMe (SOC Level 1 Path)

## Scenario
This lab introduces phishing attacks and how a SOC analyst identifies, analyzes, and responds to phishing emails using a structured investigation approach.


## Analysis Process
- Reviewed email content and sender details
- Identified suspicious indicators related to phishing
- Evaluated the risk level of the alert
- Determined the appropriate response
## Final Verdict
The analyzed email contained phishing characteristics and was treated as a potential security threat.

## Recommended Action
- User awareness
- Monitoring for similar phishing attempts



## SOC Skills Demonstrated
- Phishing awareness
- Alert triage
- Incident documentation
- Security mindset
